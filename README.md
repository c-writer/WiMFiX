# WiMFiX

WiMFiX is a 'PreTweaker' for Windows 10 & 11.

Customize windows before install.

1. Mount image (install.wim or Windows.iso)
2. Tweak it
3. Apply (make Custom_Windows.iso)
4. Make a clean install.
